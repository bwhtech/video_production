## Design guidelines for video graphics

### Current system — frappe-ui tokens (thumbnails & motion graphics)

Thumbnails and motion graphics are built in **HyperFrames** under `graphics/`
(replacing the old Remotion `~/Learning/bwh-video`). The theme is pulled from
**frappe-ui**'s espresso design tokens.

- **Token & design docs** are vendored from frappe-ui in
  **`graphics/theme/ref/`** (`frappe-ui-TOKENS.md`, `frappe-ui-DESIGN.md`,
  `frappe-ui-foundations.md`, `frappe-ui-COMPONENTS.md`) — read those for the
  token semantics and design language; don't restate them here.
- **`graphics/THEME.md`** covers the HyperFrames setup + the video-specific
  deltas: **brand = violet** (`var(--brand)`), **dark theme default**, Inter
  Variable. Vars match frappe-ui 1:1, generated by `graphics/theme/build_tokens.py`.
  Use tokens, not hex.

### Legacy palette (existing shorts-caption pipeline)

The **indigo / Roboto Mono** system below is the *original* theme, still
hardcoded in the vertical-shorts caption pipeline (`render_captions.py`,
`compose_short.py`). Leave it as-is unless migrating those to the frappe-ui
tokens above.

### Fonts

- **Primary / code:** Roboto Mono — loaded via `@remotion/google-fonts/RobotoMono`. Weights: `400`, `700`.
- **UI / supporting:** Inter — loaded via `@remotion/google-fonts/Inter`. Weights: `400`, `500`, `600`, `700`.
- **Title weight:** `800` (black) when available.
- **Design reference (Paper mocks):** Geist Mono.

### Typography / layout defaults

- `fontSize`: `40`
- `tabSize`: `3`
- `horizontalPadding`: `60`
- `verticalPadding`: `84`
